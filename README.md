# Blinkit Grocery Sales Dashboard

## Project Overview
This project is an interactive sales dashboard created using Microsoft Excel for analyzing Blinkit grocery sales data.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## Features
- Sales Analysis
- Outlet Analysis
- Item Category Analysis
- Interactive Dashboard
